# Repository Guidelines

## Purpose

このリポジトリは、ビジネスユーザーが既存業務を AI 化し、評価と iteration を通じて AI のアウトプット品質を継続的に向上させるための skills を共有する場所です。

## Current Scope

現時点では skills を実装しません。リポジトリの目的、共有範囲、今後の運用方針だけを整えます。

## Sharing Rules

- Git で共有するのは `README.md`、`skills/`、`AGENTS.md`、`.gitignore` のみです。
- `.gitignore` の許可リストを変更する場合は、共有範囲への影響を明確にしてから行います。
- 認証情報、個人情報、一時ファイル、ローカル環境固有の設定は追加しません。

## Future Skill Design

skills を追加する際は、対象業務、必要な入力、実行フロー、期待する出力、品質評価基準、iteration 方法を明確にします。
